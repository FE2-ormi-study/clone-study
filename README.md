# 📕클론코딩 스터디
 
## 스터디 계획

1. 스터디 인원 협의를 통해 1-3개의 site page 지정

2. 월-금 각자 코딩! - 야간자율학습방 을 활용해도 좋을거같습니다!

3. 각자 코드 작성하여 스터디 인원에 전체 공유

4. 토/일 하루 특정 시간에 1~2시간 정도 discord 를 통한 코드 리뷰!
